Frequently Asked Questions

ALLoy allows the rapid subtyping of Acute Lymphoblastic Leukemia (ALL) patients using ALLIUM. ALLIUM (ALL subtype Identification Using Machine learning) is a multimodal classifier of molecular subtypes in pediatric acute lymphoblastic leukemia, using DNA methylation (DNAm) and gene expression (GEX) data.

ALLoy is a Django web application that interfaces with an MLflow server running the ALLIUM model. When you upload a gene expression (GEX) file paired with a phenotype information file, this data is posted to the ALLIUM model’s inference function. The results, a set of probabilities that a particular patient belongs to the given ALL subtypes, are displayed in a formatted report. DNA methylation (DNAm) is currently being implemented, and works in a similar manner.

Download our example GEX file and example phenotype file.
